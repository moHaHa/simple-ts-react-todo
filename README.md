# React + TypeScript + Vite => Basic React ToDo App


Install Typescript (skip if you already  have it )
```bash 
npm i -g typescript

```

Install `pnpm`  (skip if you already  have it )
```bash
npm i -g pnpm 
```

Install packages

```bash
pnpm i 
```
Run The project 

```bash
pnpm run dev
```
